## ALX Shell Variables Expansions Project

### 2-path -> appends /action to the end of PATH

### 3-paths -> counts the number of directories in PATH

### 4-global_variables -> lists all environmental variables

### 5-local_variables -> lists all local and environmental variables and also functions
