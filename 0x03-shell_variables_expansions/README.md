## ALX Shell Variables Expansions Project

### 0-alias -> Creates an alias 'ls' of 'rm *'

### 1-hello_you -> prints hello user, whereby user is the current active user

### 2-path -> appends /action to the end of PATH

### 3-paths -> counts the number of directories in PATH

### 4-global_variables -> lists all environmental variables

### 5-local_variables -> lists all local and environmental variables and also functions

### 6-create_local_variable -> Creates a new local variable BEST with the value School

### 7-create_global_variable -> Creates a new global variable BEST with the value School

### 8-true_knowledge -> Prints the result of the addition of 128 and TRUEKNOWLEDGE, of which TRUEKNOWLEDGE is an environmental variable

### 9-divide_and_rule -> Prints the result of POWER divided by DIVIDE

### 10-love_exponent_breath -> Prints the result of BREATH to the power of LOVE

### 11-binary_to_decimal -> converts environmental variable BINARY to base 10

### 12-combinations -> prints all possible combinations of two letters apart from oo

### 13-print_float -> prints a number with 2 decimal places
